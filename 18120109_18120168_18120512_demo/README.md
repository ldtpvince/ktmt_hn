## Running 
Note: Calculator only processes expressions with only 1 operator 
### Qfloat:
  Các phép toán : +, -, *, /
  
### QInt:
  Các phép toán : +, -, *, /
  
  Các phép toán dịch bit: <<, >>
  
  Các phép toán quan hệ: &, |, ^, ~ (Phép NOT chưa xử lý được)
  
### Các hệ đếm
  DEX, BIN, HEX
